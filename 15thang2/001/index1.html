Thuộc tính style trong thẻ html
        *** Phân biệt thuộc tính html và thuộc tính của css
        CSS được sử dụng định dang trang trí cho giao diện html
        ***
        Thuộc tính html bổ sung tính chất ý nghĩa cho thẻ html
        Thuộc tính css được sử dụng để tạo ra 1 chức năng giao diện của css
        ví dụ màu sắc , đường viền , font chữ ...
        ***
        Thuộc tính css đầu tiên là color ( màu sắc )

        *** Khi viết mã css
        thuộc_tính_css : giá trị
        *** Nếu có nhiều thuộc tính
        thuộc_tính_css_1 : giá trị 1; thuộc_tính_css_2 : giá trị 2 ...
        Phân tách các thuộc tính bằng dấu ;

        ***
        css color : màu chữ
        css background-color: màu nền
