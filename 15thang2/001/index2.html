Thuộc tính class trong thẻ html
        được sử dụng gom nhóm các thẻ html vào 1 nhóm

        Ví dụ
        Cho các tình miền bắc màu xanh
        Miền trung màu vàng
        Miên nam màu da cam

        Nhúng css vào thông qua thẻ style có thuộc tính type="text/css"
        thẻ style nằm trong thẻ head

        *** chọn các class trong html
        trong css ta dùng .className
        ví dụ trong html class là mienbac
        thì trong css để chọn class mienbac ta dùng .mienbac

        *** Yêu cầu
        cho các tỉnh tp thuộc trung ương có màu nền là màu tím

        *** Yêu cầu
        thủ đô có font chữ to hơn các tp khác
        font-size trong css
