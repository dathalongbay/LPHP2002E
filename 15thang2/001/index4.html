Thuộc tính id trong html
        Thuộc tính class được sử dụng để gom nhóm
        Thuộc tính id được sử dụng để định danh là duy nhất
        => 1 thẻ html chỉ được phép có 1 id duy nhất
        1 thẻ html có thể có nhiều class
        trong cả 1 trang html thì không được phép có 2 thẻ trùng id
        id giống số hộ chiếu/CMT con người

        Muốn chọn thuộc tính id trong css
        thì dùng #id
        ví dụ
        #hanoi
        #hochiminh
        #danang
